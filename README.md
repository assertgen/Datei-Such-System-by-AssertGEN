## 🇩🇪 Deutsch

### Beschreibung
Das **Datei-Such-System** ist ein leistungsstarkes Desktop-Suchwerkzeug für Windows.  
Es durchsucht Word-, Excel-, PowerPoint-, PDF-, HTML-, Text- und CSV-Dateien in einem ausgewählten Ordner und allen Unterordnern – inklusive **Volltextsuche** und **Autocomplete**.

### ✨ Funktionen
- 🔍 Volltextsuche in Word, Excel, PowerPoint, PDF, HTML, TXT, CSV
- ⚡ Live-Suche mit Autocomplete (Tipp-Vorschläge)
- 📄 Dateivorschau direkt im Programm
- 📊 Öffnungsstatistik pro Datei
- 🖥 Desktop- / Startmenü- / Taskleisten-Verknüpfungen
- 🌐 Dreisprachige Benutzeroberfläche (DE / EN / TR)
- 🔒 Vollständig offline – keine Datenübertragung

### 🚀 Installation
1. `.exe`-Datei herunterladen (siehe [Releases](../../releases))
2. In einen Ordner mit Schreibrechten kopieren (z. B. `C:\Users\...\Documents\Datei-Suche\`)
3. Logo-Dateien (`assertgen-logo-uzun200.jpg`, `DR-Bad-Pyrmont-Logo.png`) in denselben Ordner legen
4. Programm starten – beim ersten Start wird ein Suchordner abgefragt

### ⚠️ Hinweis
Beim ersten Start kann eine Antivirenmeldung erscheinen. Dies ist normal bei `.exe`-Dateien, die mit PyInstaller erstellt wurden.

## 🇬🇧 English

### Description
**Datei-Such-System** is a powerful desktop search tool for Windows.  
It indexes and searches Word, Excel, PowerPoint, PDF, HTML, text and CSV files in a selected folder and all subfolders – with **full-text search** and **autocomplete**.

### ✨ Features
- 🔍 Full-text search across Word, Excel, PowerPoint, PDF, HTML, TXT, CSV
- ⚡ Live search with autocomplete suggestions
- 📄 File preview directly in the application
- 📊 Per-file open statistics
- 🖥 Desktop / Start Menu / Taskbar shortcuts
- 🌐 Trilingual interface (DE / EN / TR)
- 🔒 Fully offline – no data transmission

### 🚀 Installation
1. Download the `.exe` file (see [Releases](../../releases))
2. Copy it to a folder with write permissions (e.g. `C:\Users\...\Documents\Datei-Suche\`)
3. Place logo files (`assertgen-logo-uzun200.jpg`, `DR-Bad-Pyrmont-Logo.png`) in the same folder
4. Launch the program – on first start, you will be asked to select a search folder

### ⚠️ Note
On first launch, your antivirus software may show a warning. This is normal for `.exe` files created with PyInstaller.
